1. Extract sharepointsidekick.zip in the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Settings" In "Sharepoint Sidekick" Menu
4. Follow the on-screen instructions
